Vocabulary Grammar Listening Lessons Job

Розширені закономірності утворення слів в англійській мові з додатковими прикладами та перекладом на українську

1. Утворення ступенів порівняння прикметників:

Односкладові та двоскладові прикметники утворюють порівняльну та найвищу форму за допомогою закінчень -er та -est.

Приклади:

  • small (малий) → smaller (менший) → smallest (найменший)
  • young (молодий) → younger (молодший) → youngest (наймолодший)
  • cheap (дешевий) → cheaper (дешевший) → cheapest (найдешевший)
  • strong (сильний) → stronger (сильніший) → strongest (найсильніший)
  • cold (холодний) → colder (холодніший) → coldest (найхолодніший)

Зміна -y на -ier та -iest (коли слово закінчується на -y):

Приклади:

  • easy (легкий) → easier (легший) → easiest (найлегший)
  • funny (смішний) → funnier (смішніший) → funniest (найсмішніший)
  • heavy (важкий) → heavier (важчий) → heaviest (найважчий)
  • busy (зайнятий) → busier (більш зайнятий) → busiest (найбільш зайнятий)

Подвоєння кінцевої приголосної при додаванні суфікса:

Приклади:

  • big (великий) → bigger (більший) → biggest (найбільший)
  • hot (гарячий) → hotter (гарячіший) → hottest (найгарячіший)
  • fat (товстий) → fatter (товстіший) → fattest (найтовстіший)
  • sad (сумний) → sadder (сумніший) → saddest (найсумніший)

Багатоскладові прикметники утворюють порівняльну та найвищу форму за допомогою more та most.

Приклади:

  • beautiful (красивий) → more beautiful (красивіший) → most beautiful (найкрасивіший)
  • interesting (цікавий) → more interesting (цікавіший) → most interesting (найцікавіший)
  • comfortable (зручний) → more comfortable (зручніший) → most comfortable (найзручніший)
  • expensive (дорогий) → more expensive (дорожчий) → most expensive (найдорожчий)
  • important (важливий) → more important (важливіший) → most important (найважливіший)

Неправильні форми:

Приклади:

  • good (добрий) → better (кращий) → best (найкращий)
  • bad (поганий) → worse (гірший) → worst (найгірший)
  • far (далекий) → farther/further (дальший) → farthest/furthest (найдальший)
  • little (малий) → less (менший) → least (найменший)
  • many/much (багато) → more (більше) → most (найбільше)

2. Утворення прислівників від прикметників:

Додавання закінчення -ly до прикметників:

Приклади:

  • quick (швидкий) → quickly (швидко)
  • slow (повільний) → slowly (повільно)
  • careful (обережний) → carefully (обережно)
  • quiet (тихий) → quietly (тихо)
  • easy (легкий) → easily (легко)
  • happy (щасливий) → happily (щасливо)
  • sad (сумний) → sadly (сумно)

Зміна -y на -ily (коли прикметник закінчується на -y):

Приклади:

  • angry (злий) → angrily (злобно)
  • lazy (лінивий) → lazily (ліниво)
  • busy (зайнятий) → busily (зайнято)

Винятки та нерегулярні форми:

Приклади:

  • good (добрий) → well (добре)
  • fast (швидкий) → fast (швидко)
  • hard (твердий) → hard (тяжко)
  • late (пізній) → late (пізно)
  • early (ранній) → early (рано)

3. Утворення іменників від дієслів та прикметників:

Суфікс -ment:

Приклади:

  • develop (розвивати) → development (розвиток)
  • agree (погоджуватися) → agreement (угода)
  • enjoy (насолоджуватися) → enjoyment (насолода)
  • move (рухатися) → movement (рух)
  • appoint (призначати) → appointment (призначення)

Суфікс -tion/-sion:

Приклади:

  • inform (інформувати) → information (інформація)
  • decide (вирішувати) → decision (рішення)
  • educate (навчати) → education (освіта)
  • act (діяти) → action (дія)
  • create (створювати) → creation (створення)
  • expand (розширювати) → expansion (розширення)

Суфікс -ness:

Приклади:

  • happy (щасливий) → happiness (щастя)
  • kind (добрий) → kindness (доброта)
  • dark (темний) → darkness (темрява)
  • sad (сумний) → sadness (смуток)
  • ill (хворий) → illness (хвороба)

Суфікс -ity:

Приклади:

  • active (активний) → activity (активність)
  • possible (можливий) → possibility (можливість)
  • real (реальний) → reality (реальність)
  • electric (електричний) → electricity (електрика)
  • creative (креативний) → creativity (креативність)

4. Утворення прикметників від іменників та дієслів:

Суфікс -ful:

Приклади:

  • beauty (краса) → beautiful (красивий)
  • hope (надія) → hopeful (сповнений надії)
  • use (користь) → useful (корисний)
  • care (турбота) → careful (обережний)
  • color (колір) → colorful (барвистий)

Суфікс -less:

Приклади:

  • hope (надія) → hopeless (безнадійний)
  • home (дім) → homeless (бездомний)
  • use (користь) → useless (марний)
  • care (турбота) → careless (необережний)
  • end (кінець) → endless (безкінечний)

Суфікс -ous:

Приклади:

  • danger (небезпека) → dangerous (небезпечний)
  • fame (слава) → famous (відомий)
  • mystery (таємниця) → mysterious (таємничий)
  • mountain (гора) → mountainous (гірський)
  • poison (отрута) → poisonous (отруйний)

Суфікс -al:

Приклади:

  • nature (природа) → natural (природний)
  • origin (походження) → original (оригінальний)
  • music (музика) → musical (музичний)
  • accident (випадок) → accidental (випадковий)
  • economy (економіка) → economic (економічний)

Суфікс -ive:

Приклади:

  • attract (приваблювати) → attractive (привабливий)
  • create (створювати) → creative (креативний)
  • act (діяти) → active (активний)
  • impress (вражати) → impressive (вражаючий)
  • expense (витрати) → expensive (дорогий)

5. Утворення дієслів від іменників та прикметників:

Суфікс -ize/-ise:

Приклади:

  • modern (сучасний) → modernize (модернізувати)
  • organ (орган) → organize (організувати)
  • symbol (символ) → symbolize (символізувати)
  • apology (вибачення) → apologize (вибачатися)
  • real (реальний) → realize (усвідомлювати)

Суфікс -ify:

Приклади:

  • simple (простий) → simplify (спрощувати)
  • beauty (краса) → beautify (прикрашати)
  • pure (чистий) → purify (очищати)
  • class (клас) → classify (класифікувати)
  • intense (інтенсивний) → intensify (посилювати)

6. Утворення іменників, що позначають осіб за професією або дією:

Суфікс -er/-or:

Приклади:

  • teach (вчити) → teacher (вчитель)
  • write (писати) → writer (письменник)
  • act (діяти) → actor (актор)
  • build (будувати) → builder (будівельник)
  • paint (малювати) → painter (художник)

Суфікс -ist:

Приклади:

  • art (мистецтво) → artist (художник)
  • piano (піаніно) → pianist (піаніст)
  • science (наука) → scientist (вчений)
  • tour (тур) → tourist (турист)
  • journal (журнал) → journalist (журналіст)

Суфікс -ian:

Приклади:

  • music (музика) → musician (музикант)
  • history (історія) → historian (історик)
  • electric (електричний) → electrician (електрик)
  • politics (політика) → politician (політик)
  • library (бібліотека) → librarian (бібліотекар)

7. Утворення множини іменників:

Додавання закінчення -s:

Приклади:

  • book (книга) → books (книги)
  • car (автомобіль) → cars (автомобілі)
  • dog (собака) → dogs (собаки)
  • house (будинок) → houses (будинки)
  • computer (комп'ютер) → computers (комп'ютери)

Додавання закінчення -es (для слів, що закінчуються на -ch, -sh, -s, -x, -z):

Приклади:

  • box (коробка) → boxes (коробки)
  • bus (автобус) → buses (автобуси)
  • watch (годинник) → watches (годинники)
  • dish (страва) → dishes (страви)
  • quiz (вікторина) → quizzes (вікторини)

Заміна закінчення -y на -ies (після приголосної):

Приклади:

  • baby (дитина) → babies (діти)
  • city (місто) → cities (міста)
  • lady (леді) → ladies (леді)
  • party (вечірка) → parties (вечірки)
  • story (історія) → stories (історії)

Нерегулярні форми множини:

Приклади:

  • man (чоловік) → men (чоловіки)
  • woman (жінка) → women (жінки)
  • child (дитина) → children (діти)
  • tooth (зуб) → teeth (зуби)
  • mouse (миша) → mice (миші)
  • foot (нога) → feet (ноги)
  • sheep (вівця) → sheep (вівці)
  • fish (риба) → fish (риби)

8. Утворення минулого часу та дієприкметника минулого часу дієслів:

Правильні дієслова додають закінчення -ed:

Приклади:

  • walk (ходити) → walked (ходив)
  • play (грати) → played (грав)
  • call (дзвонити) → called (дзвонив)
  • want (хотіти) → wanted (хотів)
  • talk (говорити) → talked (говорив)

Зміна закінчення -y на -ied (після приголосної):

Приклади:

  • study (вивчати) → studied (вивчав)
  • try (спробувати) → tried (спробував)
  • copy (копіювати) → copied (копіював)
  • carry (нести) → carried (ніс)

Подвоєння кінцевої приголосної:

Приклади:

  • stop (зупинятися) → stopped (зупинився)
  • plan (планувати) → planned (планував)
  • hop (стрибати) → hopped (стрибав)
  • beg (благати) → begged (благав)

Неправильні дієслова мають особливі форми:

Приклади:

  • go (йти) → went (пішов) → gone (пішов)
  • eat (їсти) → ate (їв) → eaten (з'їв)
  • see (бачити) → saw (побачив) → seen (побачений)
  • take (брати) → took (взяв) → taken (взятий)
  • come (приходити) → came (прийшов) → come (прийшов)

9. Утворення дієприкметника теперішнього часу та герундія:

Додавання закінчення -ing:

Приклади:

  • run (бігти) → running (біг)
  • make (робити) → making (роблення)
  • swim (плавати) → swimming (плавання)
  • write (писати) → writing (писання)
  • read (читати) → reading (читання)

Подвоєння кінцевої приголосної:

Приклади:

  • sit (сидіти) → sitting (сидіння)
  • get (отримувати) → getting (отримання)
  • begin (починати) → beginning (початок)

Випадіння кінцевої -e перед -ing:

Приклади:

  • come (приходити) → coming (прихід)
  • write (писати) → writing (писання)
  • drive (водити) → driving (водіння)

10. Утворення негативних форм слів:

Префікс un-:

Приклади:

  • happy (щасливий) → unhappy (нещасливий)
  • known (відомий) → unknown (невідомий)
  • usual (звичайний) → unusual (незвичайний)
  • fair (справедливий) → unfair (несправедливий)
  • able (здатний) → unable (нездатний)

Префікс in-:

Приклади:

  • visible (видимий) → invisible (невидимий)
  • correct (правильний) → incorrect (неправильний)
  • active (активний) → inactive (неактивний)
  • dependent (залежний) → independent (незалежний)
  • efficient (ефективний) → inefficient (неефективний)

Префікс im- (перед b, m, p):

Приклади:

  • possible (можливий) → impossible (неможливий)
  • patient (терплячий) → impatient (нетерплячий)
  • mature (зрілий) → immature (незрілий)
  • moral (моральний) → immoral (аморальний)
  • perfect (досконалий) → imperfect (недосконалий)

Префікс ir- (перед r):

Приклади:

  • regular (регулярний) → irregular (нерегулярний)
  • responsible (відповідальний) → irresponsible (безвідповідальний)
  • rational (раціональний) → irrational (нераціональний)
  • relevant (релевантний) → irrelevant (нерелевантний)

Префікс il- (перед l):

Приклади:

  • legal (законний) → illegal (незаконний)
  • logical (логічний) → illogical (нелогічний)
  • literate (грамотний) → illiterate (неграмотний)

Префікс dis-:

Приклади:

  • agree (погоджуватися) → disagree (не погоджуватися)
  • appear (з'являтися) → disappear (зникати)
  • connect (підключати) → disconnect (відключати)
  • like (подобатися) → dislike (не подобатися)
  • approve (схвалювати) → disapprove (не схвалювати)

11. Утворення слів з протилежним значенням:

Префікс anti-:

Приклади:

  • social (соціальний) → antisocial (антисоціальний)
  • virus (вірус) → antivirus (антивірус)
  • bacterial (бактеріальний) → antibacterial (антибактеріальний)

Префікс de-:

Приклади:

  • compose (складати) → decompose (розкладатися)
  • activate (активувати) → deactivate (деактивувати)
  • code (кодувати) → decode (декодувати)

Префікс mis-:

Приклади:

  • understand (розуміти) → misunderstand (неправильно розуміти)
  • lead (вести) → mislead (вводити в оману)
  • behave (поводитися) → misbehave (погано поводитися)
  • spell (писати) → misspell (писати з помилками)

12. Утворення прикметників із значенням можливості або здатності:

Суфікс -able/-ible:

Приклади:

  • read (читати) → readable (читабельний)
  • access (доступ) → accessible (доступний)
  • break (ламати) → breakable (той, що ламається)
  • notice (помічати) → noticeable (помітний)
  • flex (гнути) → flexible (гнучкий)

13. Утворення іменників жіночого роду (традиційно):

Суфікс -ess:

Приклади:

  • actor (актор) → actress (актриса)
  • waiter (офіціант) → waitress (офіціантка)
  • prince (принц) → princess (принцеса)
  • host (господар) → hostess (господиня)
  • god (бог) → goddess (богиня)

14. Утворення прислівників частоти та способу дії:

Додавання суфікса -ly до прикметників:

Приклади:

  • usual (звичайний) → usually (зазвичай)
  • rare (рідкісний) → rarely (рідко)
  • frequent (частий) → frequently (часто)
  • sudden (раптовий) → suddenly (раптово)
  • quiet (тихий) → quietly (тихо)

15. Утворення станів або якостей від прикметників:

Суфікс -ness:

Приклади:

  • dark (темний) → darkness (темрява)
  • weak (слабкий) → weakness (слабкість)
  • ill (хворий) → illness (хвороба)
  • aware (обізнаний) → awareness (обізнаність)
  • lonely (самотній) → loneliness (самотність)

Висновок:

Збільшення кількості прикладів допомагає краще зрозуміти закономірності утворення слів в англійській мові. Розуміння цих правил та активне використання нових слів у мовленні та письмі сприятиме покращенню ваших знань та навичок в англійській мові.

Поради для вивчення:

  • Складайте власні речення з новими словами, щоб закріпити їх у пам'яті.
  • Практикуйте переклад з англійської на українську та навпаки.
  • Використовуйте флеш-карти або додатки для запам'ятовування нових слів та їхніх форм.
  • Читайте тексти англійською, звертаючи увагу на використання різних граматичних форм та словотвірних елементів.

Сподіваюся, ця інформація допоможе вам у вивченні англійської мови!
